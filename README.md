# tdd-felype-marinho
Implementação de um gerenciador de tarefas simples seguindo TDD.


## Projeto
Implementação de um gerenciador de tarefas simples seguindo os princípios de **TDD (Test Driven Development)**, utilizando **JavaScript (Node.js)** e o framework de testes **Jest**.

O sistema permite:
- Criar tarefas com título e descrição
- Impedir tarefas duplicadas
- Marcar tarefas como concluídas
- Remover tarefas existentes
- Listar todas as tarefas

Cada funcionalidade foi desenvolvida seguindo o ciclo **RED → GREEN → REFACTOR**.

---

## Ferramenta de testes
- **Node.js 18+**
- **Jest** (framework de testes unitários)
